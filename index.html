Soy Paula Lisseth Navarro Cárdenas, tengo 17 años y estoy dando mis primeros pasos formales en el mundo del desarrollo de software. Disfruto entender cómo funcionan las cosas por dentro: desde cómo se estructura una base de datos hasta cómo una interfaz se conecta con la lógica del servidor.

He estado formándome en introducción a la programación, Python y JavaScript, además de fundamentos de backend y frontend. También aprendí MySQL para el manejo de bases de datos relacionales y bases de inteligencia artificial, que me parecen el futuro del desarrollo.

Trabajo con Git y GitHub para el control de versiones, y entiendo cómo aplicar Scrum para organizar el trabajo en equipos de forma ágil. Mi meta es seguir creciendo como desarrolladora y aportar valor real en un equipo de tecnología.
